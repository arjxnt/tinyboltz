# TinyBoltz Roadmap

## Now

- One-command screening workflow.
- UniProt-backed target packs.
- Resumable Boltz execution.
- Ranked HTML evidence reports.
- Dependency-light CLI.

## Near Term

- Embedded 3Dmol.js pose viewer for top candidates.
- Public benchmark calibration pack.
- Vendor-neutral purchasable-compound library metadata.
- Hard budget guardrails by maximum job count or GPU-hour estimate.
- JSON and CSV report export.

## Bigger Swing

- Community-maintained neglected-disease target registry.
- Reproducible small-lab screening protocols.
- Cloud launch profiles for a single affordable GPU session.
- Calibration notebooks comparing Boltz affinity outputs to public assay data.
